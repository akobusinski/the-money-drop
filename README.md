The Money Drop
---

A project created during my freshman year of college. May contain obscure workarounds and be a tiny bit overengineered.

## Requirements
- [CMake](https://cmake.org)
- [Python 3](https://www.python.org/)
- A C++ compiler which supports C++23, I used [clang](https://clang.llvm.org/)

## Building
- Generate a build system config with CMake. CMake supports a [plethora of build systems](https://cmake.org/cmake/help/latest/manual/cmake-generators.7.html).
- Build the project with your favourite build system.
